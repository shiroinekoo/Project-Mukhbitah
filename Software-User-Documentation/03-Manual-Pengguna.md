## 3.1 Manajemen Akun

### 3.1.1 Registrasi

1. Klik **"Daftar"** di pojok kanan atas
2. Isi formulir:
   - **Nama Lengkap** — nama yang akan ditampilkan
   - **Email** — alamat email aktif
   - **Nomor Telepon** — format: 08XXXXXXXXXX
   - **Password** — minimal 8 karakter, 1 huruf kapital, 1 angka
3. Centang persetujuan Syarat & Ketentuan
4. Klik **"Daftar"**
5. Buka email → klik link verifikasi

### 3.1.2 Login

1. Klik **"Masuk"**
2. Masukkan email dan password
3. Opsional: centang **"Ingat Saya"** (sesi 30 hari)
4. Klik **"Masuk"**

**Login dengan akun sosial:**
- Klik **"Masuk dengan Google"** atau **"Masuk dengan Facebook"**
- Izinkan akses → otomatis login

### 3.1.3 Lupa Password

1. Di halaman login, klik **"Lupa Password?"**
2. Masukkan email terdaftar
3. Klik **"Kirim Link Reset"**
4. Buka email → klik link reset (berlaku 1 jam)
5. Masukkan password baru → klik **"Simpan"**

### 3.1.4 Edit Profil

1. Klik ikon 👤 → **"Akun Saya"** → **"Profil Saya"**
2. Klik **"Edit Profil"**
3. Ubah: foto profil, nama, email, atau nomor telepon
4. Klik **"Simpan"**

> **Catatan:** Mengubah email memerlukan verifikasi ulang ke email baru.

### 3.1.5 Ubah Password

1. Akun Saya → **"Ubah Password"**
2. Masukkan password lama
3. Masukkan password baru (min. 8 karakter, 1 kapital, 1 angka)
4. Konfirmasi password baru
5. Klik **"Simpan"**

### 3.1.6 Logout

- Klik ikon 👤 → **"Keluar"**
- Untuk logout semua perangkat: Akun Saya → **"Logout dari Semua Perangkat"**

## 3.2 Katalog Produk

### 3.2.1 Menjelajahi Produk

**Lewat Kategori:**
1. Klik menu **"Produk"** atau kategori di beranda
2. Pilih: Pashmina / Segi Empat / Instan

**Lewat Pencarian:**
1. Klik ikon 🔍 atau kotak pencarian
2. Ketik kata kunci (nama, material, warna)
3. Tekan Enter

### 3.2.2 Filter Produk

Di halaman produk, gunakan panel filter di sisi kiri:

| Filter | Pilihan |
|--------|---------|
| Kategori | Pashmina, Segi Empat, Instan |
| Harga | Slider rentang harga |
| Warna | Pilih satu atau lebih warna |
| Material | Voile, Ceruti, Jersey, dll. |
| Rating | 1★ hingga 5★ |
| Ketersediaan | Tersedia / Semua |

Klik **"Reset Filter"** untuk menghapus semua filter.

### 3.2.3 Mengurutkan Produk

Gunakan dropdown **"Urutkan"** di kanan atas:
- Harga: Rendah ke Tinggi
- Harga: Tinggi ke Rendah
- Terpopuler
- Terbaru
- Rating Tertinggi

### 3.2.4 Detail Produk

Klik produk untuk membuka halaman detail:

| Bagian | Isi |
|--------|-----|
| Galeri | Foto produk, klik untuk perbesar |
| 3D Viewer | Tampilan 3D interaktif |
| Info Produk | Nama, harga, diskon, rating |
| Pilihan | Warna, kuantitas |
| Tab Deskripsi | Detail lengkap produk |
| Tab Panduan Ukuran | Cara memilih ukuran yang tepat |
| Tab Perawatan | Instruksi mencuci dan menyimpan |
| Ulasan | Review dari pembeli terverifikasi |
| Produk Terkait | Rekomendasi produk serupa |

### 3.2.5 Visualisasi 3D

1. Di halaman detail produk, klik **ikon 3D**
2. **Putar:** drag dengan mouse / geser dengan jari
3. **Zoom:** scroll mouse / pinch layar sentuh
4. **Ganti warna:** klik swatch warna
5. **Reset posisi:** klik tombol reset

## 3.3 Keranjang Belanja

### 3.3.1 Menambah Produk

**Cara 1 — Dari halaman detail:**
1. Pilih warna
2. Atur jumlah (1–10)
3. Klik **"Tambah ke Keranjang"**

**Cara 2 — Quick add:**
1. Hover pada kartu produk
2. Klik ikon 🛒 yang muncul

### 3.3.2 Mengelola Keranjang

1. Klik ikon 🛒 di navbar
2. Di halaman keranjang:

| Aksi | Cara |
|------|------|
| Tambah kuantitas | Klik **+** |
| Kurangi kuantitas | Klik **-** |
| Ubah kuantitas langsung | Ketik angka di field |
| Hapus item | Klik ikon 🗑️ → konfirmasi |
| Lanjut belanja | Klik **"Lanjut Belanja"** |

### 3.3.3 Ringkasan Pesanan

Di sisi kanan halaman keranjang:
- **Subtotal** — total harga produk
- **Ongkos Kirim** — Rp 0 jika subtotal ≥ Rp 200.000
- **Diskon** — jika ada kode promo
- **Total** — jumlah yang harus dibayar

## 3.4 Proses Checkout

### 3.4.1 Langkah 1: Alamat Pengiriman

Isi atau pilih alamat:

| Field | Keterangan |
|-------|------------|
| Nama Lengkap | Nama penerima paket |
| Nomor Telepon | Nomor yang bisa dihubungi kurir |
| Alamat Lengkap | Jalan, nomor, RT/RW, kelurahan |
| Provinsi | Pilih dari dropdown |
| Kota/Kabupaten | Pilih dari dropdown |
| Kecamatan | Pilih dari dropdown |
| Kode Pos | 5 digit |
| Catatan | Patokan atau instruksi khusus (opsional) |

Centang **"Simpan alamat ini"** untuk menyimpan ke profil.

### 3.4.2 Langkah 2: Metode Pengiriman

| Kurir | Estimasi | Biaya |
|-------|----------|-------|
| JNE Regular | 3–5 hari | Rp 15.000 |
| JNE YES | 1–2 hari | Rp 25.000 |
| J&T Express | 2–4 hari | Rp 12.000 |
| SiCepat | 2–3 hari | Rp 13.000 |

> Gratis ongkir untuk semua metode jika subtotal ≥ Rp 200.000

### 3.4.3 Langkah 3: Metode Pembayaran

| Kategori | Pilihan |
|----------|---------|
| Transfer Bank | BCA, Mandiri, BNI, BRI |
| E-Wallet | GoPay, OVO, Dana, ShopeePay |
| Kartu Kredit/Debit | Visa, Mastercard, JCB |
| Gerai Retail | Indomaret, Alfamart |

**Batas waktu pembayaran:**
- Transfer bank: 24 jam
- E-wallet: 1 jam

### 3.4.4 Langkah 4: Review & Konfirmasi

1. Periksa semua detail pesanan
2. Masukkan kode promo (opsional) → klik **"Terapkan"**
3. Tambahkan catatan pesanan (opsional)
4. Centang persetujuan Syarat & Ketentuan
5. Klik **"Bayar Sekarang"**
6. Ikuti instruksi pembayaran
7. ✅ Pesanan berhasil dibuat

## 3.5 Manajemen Pesanan

### 3.5.1 Melihat Pesanan

Akun Saya → **"Pesanan Saya"**

| Status | Arti |
|--------|------|
| 🟡 Menunggu Pembayaran | Belum dibayar |
| 🔵 Diproses | Sedang dikemas |
| 🚚 Dikirim | Dalam perjalanan |
| ✅ Selesai | Sudah diterima |
| ❌ Dibatalkan | Pesanan dibatalkan |

### 3.5.2 Melacak Pesanan

1. Klik pesanan yang ingin dilacak
2. Klik **"Lacak Pesanan"**
3. Masukkan nomor resi (dikirim via email)
4. Lihat status pengiriman real-time

### 3.5.3 Membatalkan Pesanan

> Hanya bisa dibatalkan dalam **1 jam** setelah pemesanan

1. Buka detail pesanan
2. Klik **"Batalkan Pesanan"**
3. Pilih alasan pembatalan
4. Klik **"Konfirmasi Pembatalan"**

## 3.7 Ulasan Produk

### Menulis Ulasan

> Hanya untuk produk yang sudah dibeli dan pesanan berstatus "Selesai"

1. Akun Saya → Pesanan Saya → pilih pesanan selesai
2. Klik **"Tulis Review"**
3. Beri rating bintang (1–5)
4. Tulis ulasan (min. 10 karakter)
5. Upload foto (opsional, maks. 5 foto)
6. Klik **"Kirim Review"**

### Mengedit / Menghapus Ulasan

1. Akun Saya → **"Review Saya"**
2. Klik **"Edit"** atau **"Hapus"** pada ulasan
