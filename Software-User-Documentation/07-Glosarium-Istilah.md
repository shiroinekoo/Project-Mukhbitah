# 7. Glosarium dan Istilah
## Mukhbitah - Platform Gallery Hijab

**Version:** 1.0  
**Date:** April 2, 2026  
**Untuk:** Semua Pengguna

---

## 7.1 Istilah Produk

| Istilah | Definisi |
|---------|----------|
| **Hijab** | Penutup kepala yang dikenakan oleh wanita Muslim sebagai bagian dari busana syar'i |
| **Pashmina** | Jenis hijab berbentuk persegi panjang dengan bahan lembut dan ringan, biasanya berukuran 75×180 cm. Cocok untuk berbagai gaya pemakaian |
| **Segi Empat** | Hijab berbentuk persegi (biasanya 115×115 cm) yang serbaguna dan bisa dipakai dengan berbagai cara |
| **Instan** | Hijab siap pakai yang sudah dijahit sedemikian rupa sehingga mudah dan cepat dipakai tanpa perlu melipat atau menyematkan |
| **Voile** | Bahan kain tipis, ringan, dan sedikit transparan. Cocok untuk cuaca panas |
| **Ceruti** | Bahan kain dengan tekstur halus, jatuh, dan sedikit mengkilap. Memberikan kesan elegan |
| **Jersey** | Bahan kain rajut yang elastis dan nyaman dipakai. Tidak mudah kusut |
| **Sifon** | Bahan kain tipis, ringan, dan mengalir. Memberikan kesan feminin dan anggun |
| **Wolfis** | Singkatan dari "Wolly Crepe". Bahan tebal, tidak transparan, dan tidak mudah kusut |
| **Motif** | Pola atau desain yang terdapat pada kain hijab (polos, bunga, abstrak, dll.) |
| **Warna Nude** | Warna-warna netral yang mendekati warna kulit, seperti krem, beige, dan coklat muda |

## 7.2 Istilah Belanja

| Istilah | Definisi |
|---------|----------|
| **Keranjang Belanja** | Fitur untuk menyimpan produk yang ingin dibeli sebelum melakukan checkout |
| **Wishlist** | Daftar produk yang disimpan untuk dibeli di kemudian hari, tanpa langsung memasukkan ke keranjang |
| **Checkout** | Proses menyelesaikan pembelian, mulai dari pengisian alamat hingga pembayaran |
| **Subtotal** | Total harga produk sebelum ditambah ongkos kirim dan dikurangi diskon |
| **Ongkos Kirim (Ongkir)** | Biaya pengiriman paket dari gudang ke alamat pembeli |
| **Gratis Ongkir** | Kebijakan di mana biaya pengiriman ditanggung oleh Mukhbitah untuk pembelian ≥ Rp 200.000 |
| **Kode Promo** | Kode unik yang dapat dimasukkan saat checkout untuk mendapatkan diskon |
| **Diskon** | Pengurangan harga dari harga normal |
| **Flash Sale** | Penjualan dengan diskon besar dalam waktu terbatas |
| **Stok** | Jumlah produk yang tersedia untuk dibeli |
| **Stok Habis** | Kondisi di mana produk tidak tersedia untuk dibeli sementara |
| **Pre-order** | Pemesanan produk yang belum tersedia, akan dikirim setelah produk siap |

## 7.3 Istilah Pembayaran

| Istilah | Definisi |
|---------|----------|
| **Payment Gateway** | Layanan yang memproses transaksi pembayaran online secara aman (Midtrans, Stripe) |
| **Transfer Bank** | Metode pembayaran dengan mentransfer uang dari rekening bank ke rekening tujuan |
| **Virtual Account (VA)** | Nomor rekening virtual unik yang dibuat untuk setiap transaksi, memudahkan verifikasi pembayaran otomatis |
| **E-Wallet** | Dompet digital untuk menyimpan dan menggunakan uang secara elektronik (GoPay, OVO, Dana, ShopeePay) |
| **Kartu Kredit** | Kartu pembayaran yang memungkinkan pembelian dengan sistem kredit/cicilan |
| **Kartu Debit** | Kartu pembayaran yang langsung memotong saldo rekening tabungan |
| **Cicilan 0%** | Pembayaran dengan kartu kredit yang dibagi menjadi beberapa bulan tanpa bunga tambahan |
| **Tokenisasi** | Proses mengganti data kartu kredit sensitif dengan token unik untuk keamanan |
| **Refund** | Pengembalian uang kepada pembeli setelah pembatalan atau pengembalian produk |
| **Konfirmasi Pembayaran** | Proses verifikasi bahwa pembayaran telah diterima oleh sistem |

## 7.4 Istilah Pengiriman

| Istilah | Definisi |
|---------|----------|
| **Kurir** | Perusahaan jasa pengiriman paket (JNE, J&T, SiCepat, dll.) |
| **Nomor Resi** | Kode unik untuk melacak status pengiriman paket |
| **Tracking** | Pelacakan posisi dan status paket secara real-time |
| **Estimasi Pengiriman** | Perkiraan waktu paket tiba di tujuan (dalam hari kerja) |
| **Hari Kerja** | Hari Senin hingga Jumat, tidak termasuk hari libur nasional |
| **COD (Cash on Delivery)** | Metode pembayaran di mana pembeli membayar saat paket tiba (belum tersedia di Mukhbitah) |
| **Manifest** | Dokumen yang berisi daftar paket yang akan dikirim |
| **Retur** | Pengembalian paket dari pembeli ke penjual |

## 7.5 Istilah Akun & Keamanan

| Istilah | Definisi |
|---------|----------|
| **Akun** | Profil pengguna yang berisi informasi pribadi, riwayat pesanan, dan preferensi |
| **Login / Masuk** | Proses autentikasi untuk mengakses akun dengan email dan password |
| **Logout / Keluar** | Proses mengakhiri sesi login |
| **Verifikasi Email** | Proses konfirmasi bahwa alamat email yang didaftarkan valid dan milik pengguna |
| **Password** | Kata sandi rahasia untuk melindungi akun |
| **Reset Password** | Proses mengganti password yang terlupa melalui email |
| **Sesi** | Periode waktu aktif setelah login hingga logout atau kedaluwarsa |
| **Enkripsi** | Proses mengamankan data agar tidak bisa dibaca oleh pihak yang tidak berwenang |
| **SSL/TLS** | Protokol keamanan yang mengenkripsi data yang dikirim antara browser dan server |
| **RBAC** | Role-Based Access Control — sistem yang membatasi akses berdasarkan peran pengguna (pelanggan/admin) |

## 7.6 Istilah Teknis Website

| Istilah | Definisi |
|---------|----------|
| **Browser** | Aplikasi untuk mengakses internet (Chrome, Firefox, Safari, Edge) |
| **Cache** | Data yang disimpan sementara di browser untuk mempercepat loading halaman |
| **Cookies** | File kecil yang disimpan browser untuk mengingat preferensi dan sesi pengguna |
| **JavaScript** | Bahasa pemrograman yang membuat website interaktif dan dinamis |
| **Responsive Design** | Desain website yang menyesuaikan tampilan dengan ukuran layar perangkat |
| **3D Viewer** | Fitur tampilan tiga dimensi interaktif untuk melihat produk dari berbagai sudut |
| **Glassmorphism** | Gaya desain UI dengan efek kaca buram yang transparan |
| **CDN** | Content Delivery Network — jaringan server yang mendistribusikan konten agar website lebih cepat |
| **API** | Application Programming Interface — cara sistem berkomunikasi satu sama lain |
| **Webhook** | Notifikasi otomatis yang dikirim sistem saat terjadi event tertentu (misal: pembayaran berhasil) |
