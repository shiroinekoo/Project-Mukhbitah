# 🚀 Upload ke GitHub - shiroinekoo/Mukhbitah

## Perintah Git untuk Repository Anda

Jalankan perintah berikut di terminal (pastikan Anda berada di folder `mukhbitah`):

```bash
# 1. Initialize git
git init

# 2. Add all files
git add .

# 3. First commit
git commit -m "Initial commit: Mukhbitah hijab gallery website"

# 4. Add remote repository
git remote add origin https://github.com/shiroinekoo/Mukhbitah.git

# 5. Rename branch to main
git branch -M main

# 6. Push to GitHub
git push -u origin main
```

## ✅ Selesai!

Setelah berhasil, buka: https://github.com/shiroinekoo/Mukhbitah

---

## 🔄 Update Setelah Perubahan

Setiap kali ada perubahan:

```bash
git add .
git commit -m "Deskripsi perubahan"
git push
```

## 🔑 Jika Diminta Password

Gunakan **Personal Access Token**:
1. Buka: https://github.com/settings/tokens
2. Generate new token (classic)
3. Pilih scope: **repo**
4. Copy token dan gunakan sebagai password
