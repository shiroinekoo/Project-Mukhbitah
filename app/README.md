## 🚀 Tech Stack

- **Framework**: Next.js 16.2.4 (App Router with Turbopack)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion 11.15.0
- **Icons**: Lucide React 0.469.0
- **Fonts**: Playfair Display (headings), Poppins (body)

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/yourusername/mukhbitah.git

# Masuk ke folder project
cd mukhbitah

# Install dependencies
npm install --legacy-peer-deps

# Run development server
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

> **Note**: Gunakan `--legacy-peer-deps` karena React 19 compatibility dengan beberapa dependencies.

```
│   ├── CategorySection.tsx # Category cards
│   ├── FeaturedProducts.tsx # Featured products grid
│   ├── ProductCard.tsx     # Reusable product card
│   ├── TestimonialSection.tsx # Customer testimonials
│   └── CTASection.tsx      # Newsletter CTA
└── tailwind.config.ts      # Tailwind configuration
```
